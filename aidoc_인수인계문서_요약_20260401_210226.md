## AiDoc 인수인계 문서 요약

### 개요
AiDoc은 기업 내 대규모 문서를 AI 기반으로 자동 관리하고 사용자 맞춤 추천을 제공하는 엔터프라이즈 시스템이다. SentenceBERT, 클러스터링, 딥러닝 모델 등을 활용해 문서를 분석하고 추천한다.

### 주요 구성
- flaskapi: REST API 서버, 모든 요청의 진입점
- embedding: SentenceBERT 기반 문서 벡터화
- category: BIRCH + XGBoost 기반 자동 분류
- nextdoc: GRU/LSTM 기반 다음 문서 예측
- userprofile: Triplet Loss 기반 사용자 프로필 학습
- relatedkeyword: FastText 기반 키워드 추출
- cycle: 사용자 로그 기반 주기적 추천
- expert: 전문가 추천 기능
- utils: 공통 유틸리티

### 기술 및 설정
- config/conf.ini 중심 설정 관리
- 도메인별 JSON 설정 파일 별도 존재

### 데이터 흐름
- 신규 문서 → 벡터화 → 분류 → 추천 생성
- 사용자 행동 로그 기반 개인화 추천 수행

### 결론
AiDoc은 다양한 AI/NLP 기술을 결합하여 기업 문서 관리와 추천을 자동화하며, 모듈형 구조로 확장성과 다양한 산업 적용이 가능하다.
