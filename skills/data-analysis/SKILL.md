# Data Analysis Skill

## 언제 사용하나
- 수치 계산이나 통계 처리가 필요할 때
- CSV/데이터 파일을 파이썬으로 처리해야 할 때
- 알고리즘 검증, 수식 계산, 데이터 변환이 필요할 때
- "계산", "통계", "평균", "합계", "분석", "코드 실행" 키워드가 포함될 때

## 작업 순서
1. 요청을 분석해 필요한 계산/처리 파악
2. Python 코드 작성 (간결하고 명확하게)
3. `run_code`로 실행
4. 결과를 해석해 사용자 언어로 설명

## 코드 작성 원칙
- 단순 계산은 한 번에 처리
- 복잡한 처리는 단계별로 분리해 실행
- 결과값은 반드시 print()로 출력하거나 변수 `result`에 저장
- 외부 라이브러리 필요 시 import 포함 (numpy, pandas, statistics 등)

## 출력 형식
- 계산 결과를 명확한 수치로 제시
- 단위 및 의미 설명
- 필요시 해석/시사점 추가

## 주의사항
- 코드 실행 전 로직을 검토해 오류 최소화
- 실행 오류 발생 시 코드 수정 후 재실행
- 대용량 데이터는 청크 단위로 처리
