# History Files Skill

## 언제 사용하나
- "처음에 첨부한 파일이 뭐야?", "이전 파일 정보 알려줘" 등 이전 대화의 첨부 파일을 참조할 때
- 현재 메시지에 파일이 없지만 과거 첨부 파일에 대해 묻는 경우

## 작업 순서
1. 시스템 프롬프트의 `<agent_memory>` 섹션(단기 메모리)에서 이전 대화 첨부 파일 목록 확인
2. 몇 번째 턴의 파일인지 파악 (사용자가 "처음", "두 번째" 등으로 언급한 경우 해당 턴 선택)
3. 파일 경로를 확인한 뒤 필요에 따라 `load_file_content` 또는 `rag_search` 호출
4. 파일이 현재 메시지에 없더라도 단기 메모리에 경로가 있으면 도구를 통해 접근 가능

## 주의사항
- 단기 메모리를 확인하기 전에 "파일이 없다"고 단정하지 말 것
- 사용자가 "처음"이라고 하면 단기 메모리의 첫 번째 사용자 턴 파일만 대상으로 할 것
- 불필요하게 모든 첨부 파일을 대상으로 작업을 수행하지 말 것
