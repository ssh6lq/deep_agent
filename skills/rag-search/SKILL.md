# RAG Search Skill

## 언제 사용하나
- 사용자가 첨부한 파일에서 특정 정보를 찾아야 할 때
- "이 파일에서", "문서에서 찾아줘", "첨부 파일 기준으로", "~에 대한 내용" 키워드가 포함될 때
- 파일 전체를 요약하는 것이 아니라 특정 부분을 검색해야 할 때

## file-analysis와의 차이
| 상황 | 사용 skill |
|------|-----------|
| 파일 전체 요약/분석/번역 | file-analysis |
| 파일에서 특정 내용 검색·질의응답 | rag-search (이 skill) |

## 작업 순서
1. 사용자의 질문에서 핵심 검색어 추출
2. `rag_search(query)` 호출
3. 반환된 청크에서 관련 내용 파악
4. 원문 근거를 포함해 답변 작성

## 출력 형식
- 답변 내용 (명확하고 직접적으로)
- 근거가 된 문서 구절 인용 (필요 시)
- 해당 내용이 없으면 "첨부 파일에서 관련 내용을 찾을 수 없습니다" 명시

## 주의사항
- 파일이 첨부되지 않은 경우 rag_search 미사용
- 검색 결과가 불충분하면 다른 검색어로 재시도
- 파일에 없는 내용은 추측하지 말 것
