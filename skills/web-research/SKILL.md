# Web Research Skill

## 언제 사용하나
- 사용자가 특정 주제에 대한 최신 정보, 뉴스, 트렌드를 요청할 때
- 사실 확인이 필요하거나 다양한 출처에서 정보를 수집해야 할 때
- "검색", "조사", "알아봐줘", "최신", "트렌드" 키워드가 포함된 요청

## 작업 순서
1. 요청을 분석해 핵심 검색 키워드 3~5개 도출
2. `internet_search`로 각 키워드별 검색 실행
3. 수집된 결과를 교차 검증하고 중복 제거
4. 핵심 내용을 구조화된 형식으로 정리

## 출력 형식
- 핵심 발견 사항을 bullet point로 정리
- 각 항목에 출처(URL) 명시
- 신뢰도가 낮거나 상충되는 정보는 별도 표시
- 마지막에 종합 요약 1~2문장

## 주의사항
- 단일 검색으로 충분하지 않으면 후속 검색어로 심화 조사
- 최신성이 중요한 경우 topic="news" 파라미터 활용
